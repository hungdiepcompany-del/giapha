# Tổng quan dự án WEB GIA PHẢ

## Mục tiêu

- Làm web gia phả dùng lâu dài.
- Có UI thêm/sửa/xóa thành viên.
- Có cây gia phả tương tác.
- Có thể chỉnh sửa trực tiếp trên cây.
- Có phân quyền.
- Có lịch sử chỉnh sửa.
- Có public/private mode.
- Có export JSON/GEDCOM/ZIP từ đầu.

## Người dùng chính

- OWNER
- ADMIN
- EDITOR
- CONTRIBUTOR
- FAMILY_VIEWER
- PUBLIC_VIEWER

## Nguyên tắc 100 năm

- Dữ liệu phải xuất được.
- Không phụ thuộc duy nhất vào cloud/database.
- Không xóa cứng.
- ID ổn định.
- Backup có manifest/checksum.
